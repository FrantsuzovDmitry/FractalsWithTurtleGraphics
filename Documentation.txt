В этом проекте я использовал закрытую для редактирования стороннюю графическую библиотеку https://github.com/nakov/TurtleGraphics.NET.git

Программа получает на вход некоторые условия, и на основе полученных аксиом (правил) рисует фракталы.
По умолчанию доступно лишь 4 поля для ввода правил, но если необходимо больше, их можно легко добавить (фактически, копипастом)

Класс MyTurtle - обертка над статическим классом Turtle из внешней библиотеки.
Класс TurtleGraphicsExtention - класс расширения для MyTurtle, который хранит в себе стек состояний черепашки.
Form1 - форма для взаимодействия пользователя с приложением.

С увеличением числа итераций фигура может уходить за пределы экрана - в некоторых случаях вы можете просто растянуть рабочее окно.
Изменить начальное положение черепашки можно в MyTurtle.InitializeTurtle, задав Angle или, при необходимости, убрав смещение.
Также можно поиграться с длиной линий у черепашки для получения более чистого результата.
